## Run Locally

Clone the project

```bash
  git clone https://github.com/bogdanblare/odin-mini-message-board-main.git
```

Go to the project directory

```bash
  cd odin-mini-message-board-main
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
